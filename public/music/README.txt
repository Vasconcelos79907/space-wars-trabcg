SPACE WARS — Music Tracks
=========================
Place the following files in this folder:

  Menu_Soundtrack.mp3   → Loops continuously in all menus
  Wave1.mp3             → Waves 1, 6, 11, 16... (first wave of each cycle)
  Wave2.mp3             → Waves 2, 7, 12, 17...
  Wave3.mp3             → Waves 3, 8, 13, 18...
  Wave4.mp3             → Waves 4, 9, 14, 19... (last wave before boss)
  Boss1.mp3             → Boss encounters 1, 3, 5... (odd bosses: waves 5,15,25...)
  Boss2.mp3             → Boss encounters 2, 4, 6... (even bosses: waves 10,20,30...)

RULES:
- Music transitions happen immediately when enemies are cleared (no silence)
- Boss music starts as soon as the wave before the boss ends
- Menu music never restarts when switching between menu screens
- All tracks loop until the next track takes over
- Supported formats: .mp3  (recommended)
